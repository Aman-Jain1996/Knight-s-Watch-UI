Inputs

Input fields come into picture in the webpage where we require any information from the user and save it for futher use.

Input Text with labels

Input fields are most used type of input field to get some data from user and they usually come with some labels.

Class-Usage : input-group | input-group err

Wrong Password

                        
                <div class="input-group">
                    <label class="label" for="name">Username </label>
                    <input type="text" name="name" id="name">
                </div>


                <div class="input-group err">
                    <label class="label" for="password">Password</label>
                    <input type="text" name="password" id="password">
                    <p class="error-message">Wrong Password</p>
                </div>

Radio buttons

Radio buttons allow the user to select one value from a set values. In this input fields , the below classes are added in HTML itself without any class

Class-Usage : checked | disabled 

                        
                    <label for="radio-1">
                        <input type="radio" name="radio" id="radio-1">
                        Radio 1
                    </label>

                    <label for="radio-2" >
                        <input type="radio" name="radio" id="radio-2" checked>
                        Radio 2
                    </label>

                    <label for="radio-3" >
                        <input type="radio" name="radio" id="radio-3" disabled>
                        Radio 3
                    </label>

Checkbox

Checkboxes allow the user to select one or more items from a set.

Class-Usage : checked | disabled 

                        
                <label for="checkbox-1">
                    <input type="checkbox" name="checkbox" id="checkbox-1">
                    Checkbox 1
                </label>

                <label for="checkbox-2" >
                    <input type="checkbox" name="checkbox" id="checkbox-2" checked>
                    Checkbox 2
                </label>
                
                <label for="checkbox-3" >
                    <input type="checkbox" name="checkbox" id="checkbox-3" disabled>
                    Checkbox 3
                </label>
