Navigation

Navigation component can be used inside 'nav' tag to give redirection to different pages with-in / out of website.

Simple Nav

This type of navigation just have diffrent links within it that are equally apart.

Class-Usage : nav

Nav Link-1 Nav Link-2 Nav Link-3 Nav Link-4 
                        
                        <div class="nav">

                            <span><a href="">Nav Link-1</a></span>

                            <span><a href="">Nav Link-2</a></span>

                            <span><a href="">Nav Link-3</a></span>

                            <span><a href="">Nav Link-4</a></span>

                        </div>

Dropdown Nav

This type of navigation shows up when an element in nav is hovered.

Note : Hover over 'Nav Link-2' to see dropdown navigation.

Class-Usage : hover-nav

Nav Link-1 Nav Link-2
Nav Link-2.i Nav Link-2.ii Nav Link-2.iii
Nav Link-3 Nav Link-4 
                        
                        <div class="nav">

                            <span><a href="">Nav Link-1</a></span>

                            <span class="hover-element">
                                <a href="" >Nav Link-2</a>

                                <!-- hover-nav -->
                                
                                <div class="hover-nav">

                                    <span>
                                        <a href="">Nav Link-2.i</a>
                                    </span>

                                    <span>
                                        <a href="">Nav Link-2.ii</a>
                                    </span>

                                    <span>
                                        <a href="">Nav Link-2.iii</a>
                                    </span>

                                </div>

                            </span>

                            <span><a href="">Nav Link-3</a></span>

                            <span><a href="">Nav Link-4</a></span>
                        </div>
