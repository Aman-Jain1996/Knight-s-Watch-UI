Alert

An alert is used to display an important message in a way that attracts the user's attention and it provides contextual feedback messages for typical user actions with the handful alert messages.

Simple Alert

Simple alerts are used to display an important message. They can be created using 'avatar' class and can be clubbed with different variants for variations.

Variant-Usage : primary | danger | success | warning | secondary

Primary! This is a Primary Alert - Check it now!
Danger! This is a Danger Alert - Check it now!
Success! This is a Success Alert - Check it now!
Warning! This is a Warning Alert - Check it now!
Secondary! This is a Secondary Alert - Check it now! 
                        
        <div class="alert primary">
            <strong>Primary! </strong>This is a Primary Alert - Check it now!
        </div>
                            
        <div class="alert danger">
            <strong>Danger!</strong> This is a Danger Alert - Check it now!
        </div>
                            
        <div class="alert success">
            <strong>Success!</strong> This is a Success Alert - Check it now!
        </div>
                            
        <div class="alert warning">
            <strong>Warning!</strong> This is a Warning Alert - Check it now!
        </div>
                            
        <div class="alert secondary">
            <strong>Secondary!</strong> This is a Secondary Alert - Check it now!
        </div>
