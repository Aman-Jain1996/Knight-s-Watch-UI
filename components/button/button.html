Buttons

Buttons are the soul of an interactive Web Application. Buttons allow users to take actions on UI and make choices with a single tap.

Default Button

They can be used using 'btn' class and we can also have variants of buttons using different variant classes.

Class-Usage : btn-primary | -secondary | -success | -error 

                        
                        <button class="btn">Default</button>

                        <button class="btn btn-primary">Primary</button>

                        <button class="btn btn-secondary">Secondary</button>

                        <button class="btn btn-success">Success</button>

                        <button class="btn btn-error">Error</button>

Outlined Button

Outlined buttons don't have any background they just have borders of different colors based on variations.

Class-Usage : outline-default | -primary | -secondary | -success | -error 

                        
                        <button class="btn outline-default">Default</button>

                        <button class="btn outline-primary">Primary</button>

                        <button class="btn outline-secondary">Secondary</button>

                        <button class="btn outline-success">Success</button>

                        <button class="btn outline-error">Error</button>

Rounded Button

Rounded buttons are usually rounded from all their corners.

Class-Usage : round-sm (4px) | -m (6px) | -lg (8px) | -xl (10px) 

                        
                <button class="btn">Default</button>

                <button class="btn outline-primary rounded-sm">Primary</button>

                <button class="btn btn-secondary rounded-m">Secondary</button>

                <button class="btn outline-success rounded-lg">Success</button>
                
                <button class="btn btn-error rounded-xl">Error</button>

Button with a hover state

Rounded buttons are usually rounded from all their corners. Hover mouse on the below buttons to see the change.

Class-Usage : btn-hover 

                        
        <button class="btn btn-hover">Default</button>

        <button class="btn outline-primary rounded-sm btn-hover">Primary</button>

        <button class="btn btn-secondary rounded-m btn-hover">Secondary</button>

        <button class="btn outline-success rounded-lg btn-hover">Success</button>
        
        <button class="btn btn-error rounded-xl btn-hover">Error</button>

Icon Button

Icon Buttons can be created by adding icon as a child to parent button. Use the 'btn' class followed by 'btn-icon'.

Class-Usage : btn-icon 

                        
                            <button class="btn btn-icon">
                                <ion-icon name="save"></ion-icon>
                                Default
                            </button>

                            
                            <button class="btn btn-primary btn-icon">
                                <ion-icon name="arrow-down"></ion-icon>
                                Primary
                            </button>
