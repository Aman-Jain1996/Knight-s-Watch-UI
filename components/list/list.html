Nested Lists

Nested lists are usually used for navigation purpose in a webpage

Class-Usage : 'list' with 'list-nested'

  • Category 1
  • Category 2

  • Category 3

    • Category 3-i
    • Category 3-ii
    • Category 3-iii
  • Category 4
                        
                        <ul class="list">

                            <label for="" class="list-heading" >Categories</label>
                            <li>Category 1</li>
                            <li>Category 2</li>

                            <li class="list-nested">
                                <p>Category 3</p>

                                <ul>
                                    <li>Category 3-i</li>
                                    <li>Category 3-ii</li>
                                    <li>Category 3-iii</li>
                                </ul>

                            </li>

                        </ul>

Select List Group

Select list group can be used for filtering data in a webpage based on checked values.

Class-Usage : We will be use 'list' along with 'checkbox label' 

                        
                        <ul class="list">
                            <label for="" class="list-heading">Categories</label>

                            <label for="checkbox">
                                <input type="checkbox" checked>
                                <li>Category 1</li>
                            </label>

                            <label for="checkbox">
                                <input type="checkbox" checked>
                                <li>Category 2</li>
                            </label>

                            <label for="checkbox">
                                <input type="checkbox" checked>
                                <li>Category 3</li>
                            </label>
                        </ul>
