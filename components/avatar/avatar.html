Avatar

Avatars are usually used as profile pictures in any website or can be clubbed to show user availability status. They are usually circular in shape. Avatars can be used to give recognisable visual identity.

Sized Avatars

These are the different sizes of avatars that can be used in your website.

Class-Usage : sm (4rem) | m (5rem) | lg (7rem) | xl (10rem) 

                        
                    <img class="sm" src="imgage-source" alt=" Badge 1">

                    <img class="m" src="imgage-source" alt="Badge 2">

                    <img class="lg" src="imgage-source" alt="Badge 3">
                    
                    <img class="xl" src="imgage-source" alt="Badge 4">

Profile Avatars

Profile Avatars can be created using the class 'avatar'. You can also size them using sized class as used above.

Class-Usage : avatar 

                        
                        <img class="avatar" src="image-source" alt="Badge 1">

                        <img class="avatar sm" src="image-source" alt="Badge 2">

                        <img class="avatar lg" src="image-source" alt="Badge 3">

Square Avatars

Square Avatars are rounded from their each corners.

Class-Usage : square 

                        
                <img class="avatar square" src="image-source" alt="Badge 1">

                <img class="avatar square sm" src="image-source" alt="Badge 2">

                <img class="avatar square lg" src="image-source" alt="Badge 3">

Stacked Avatars

Stacked Avatars are used to club more than 1 avatars as an single representation.

Class-Usage : stacked 

                        
                            <!-- Wrap thses elements inside a container with class 'stacked' -->

                <div class="stacked">
                    <img class="avatar" src="https://picsum.photos/id/237/200/300" 
                    alt="Badge">

                    <img class=" avatar" src="https://picsum.photos/id/1005/200/300" 
                    alt="Badge">

                    <img class="avatar" src="https://picsum.photos/id/1012/200/300" 
                    alt="Badge">

                </div>

Status Avatars

These avatars are used to show availability of the user.

Class-Usage : busy | available | offline 

                        
                <div class="busy">
                    <img class="avatar" src="image-source" alt="Badge">
                </div>


                <div class="available">
                    <img class="avatar sm" src="image-source" alt="Badge">
                </div>
                

                <div class="offline">
                    <img class="avatar square lg" src="image-source" alt="Badge">
                </div>
