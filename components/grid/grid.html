Grids

A grid is a set of intersecting horizontal and vertical lines defining columns and rows. Elements can be placed onto the grid within these column and row lines. Grids adapt to screen size and orientation, ensuring consistency across layouts.

Grid 2 column

This grid type has 2 columns and on adding more than 2 items,new item will occupy new row.

Class-Usage : grid-2-col 

                        
                <div class="grid grid-2-col">

                        <img src="https://picsum.photos/id/110/600/400" alt="sunset" class="img-responsive" />
                            
                        <img src="https://picsum.photos/id/110/600/400" alt="sunset" class="img-responsive"/>
                            
                        <img src="https://picsum.photos/id/110/600/400" alt="sunset" class="img-responsive"/>
                </div>
