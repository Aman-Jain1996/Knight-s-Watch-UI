Introduction to Knight's Watch

Knight's Watch is an open source styling library, which helps Developers to use best styles and components to create their styling by using pre-defined classnames. It helps you to quickly build your Dream Project, by focusing more on Functionality and leave your design worries on us.

Installation Guide

In order to use Knight's Watch components in your project, Copy the link given below and paste it in the <head> tag of your html to get started with the classes.

Customizing Themes

Although all your styling issues have been taken care off and all the best practices are followed , but we respect our User's need and all the components in this library are made keeping this is mind. There are times, when the pre-defined styles are not sufficient

In order to override the predefined base classes, users can add their own class to the pre-existing classes, and style them seperately, as shown below. 


                    <!-- Default styling from the button component -->

                    <button class="btn">Custom Button</button>


                    <!-- Overriding styling using custom classes-->

                    <style>

                         .custom-style {
                            background-color: aquamarine;
                            color:black;
                        }

                    </style>
